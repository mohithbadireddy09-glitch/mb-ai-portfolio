# AI Content Generator

This project demonstrates how AI can help businesses generate content for marketing and social media.

## Purpose
To create structured content ideas and draft posts using AI.

## Workflow
1. Enter the business type or niche
2. Enter the topic
3. AI generates content ideas
4. AI drafts a post or caption

## Tools
- ChatGPT
- Prompt engineering
- Content structuring
