# AI Recruitment Helper

## Purpose
This project demonstrates how AI can help HR teams screen and evaluate job applications.

## Problem
Recruiters often receive hundreds of CVs and manually reviewing them takes significant time.

## AI Workflow
1. Recruiter uploads resumes
2. AI scans skills, experience, and keywords
3. AI ranks candidates based on job requirements
4. Recruiter reviews the shortlisted applicants

## Tools Used
ChatGPT
Prompt engineering
Basic HR screening logic

## Example Use Case
Screen 100 CVs for a marketing assistant role and shortlist the top 10 candidates based on relevant skills and experience.
