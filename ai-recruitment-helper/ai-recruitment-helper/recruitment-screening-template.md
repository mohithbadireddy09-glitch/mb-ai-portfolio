# Recruitment Screening Prompt Template

Use this prompt with AI to evaluate candidates for a role.

## Input

Job Title:  
Key Skills Required:  
Experience Level:  
Number of CVs to Screen:  

## Prompt

Act as an HR screening assistant.

Review the candidate profiles based on the following:
- relevance of skills
- years of experience
- role fit
- communication or transferable strengths

Return:
1. Top candidates
2. Matching skills
3. Missing skills
4. Overall recommendation

## Output Format

- Candidate Name
- Match Score
- Strengths
- Gaps
- Final Recommendation
