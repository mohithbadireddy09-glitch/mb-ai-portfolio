# AI Recruitment Helper

This project demonstrates how AI can assist recruiters in screening job candidates efficiently.

## Purpose
To help analyze CVs and shortlist candidates using AI prompts.

## Workflow
1. Paste the job description
2. Paste the candidate CV
3. AI evaluates skill match and experience
4. AI provides recommendation

## Tools
- ChatGPT
- Prompt engineering
- Recruitment workflow analysis
